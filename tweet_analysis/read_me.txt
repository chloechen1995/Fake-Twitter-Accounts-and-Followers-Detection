This folder contains the following scripts:
csv_header.py -- Change the headers of csv files
final_tweet_df.py -- Create a csv file that contains the tweet analysis results
left_id.py -- Create a csv file that stores twitter_ids that do not get tweet information using the tweet.py due to the API Restriction
recent_tweet.py -- select the most recent 200 tweets 
similarity_1.py -- calculate tweet similarity: $S=\\frac{\\sum_{p\\in P}c(p)}{l_al_p}$
