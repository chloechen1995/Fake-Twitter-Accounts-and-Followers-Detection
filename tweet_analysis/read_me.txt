This folder contains the following scripts:
Main Scripts:
random_forest.py - builds random forest model
combine_df.py - combines all the feature results into a dataframe
tweet_time_diff.py - calculates the time difference between two consecutive tweets
tweet_ratio.py - creates a csv file that contains the tweet analysis ratios
similarity_1.py - calculates tweet similarity ratio 1
similarity_2.py - calculates tweet similarity ratio 2
sim_1_df.py - creates a dataframe for storing the tweet similarity ratio 1 result
sim_2_df.py - creates a dataframe for storing the tweet similarity ratio 2 result

Secondary Scripts
csv_header.py -- Change the headers of csv files
final_tweet_df.py -- Create a csv file that contains the tweet analysis results
left_id.py -- Create a csv file that stores twitter_ids that do not get tweet information using the tweet.py due to the API Restriction
recent_tweet.py -- select the most recent 200 tweets 
tweet_left.py -- get tweets for the leftover twitter_ids from API
tweet_time.py -- calculate the time difference between two consecutive tweets
